## The fast way

Tell Claude (or any MCP-compatible AI):

```
go to slashvibe.dev and install /vibe
```

Claude reads the install page, downloads the MCP server config, and sets it up. You're done.

## Manual install

If you prefer to set it up yourself:

<Steps>
  <Step title="Sign in with GitHub">
    Go to [slashvibe.dev](https://slashvibe.dev) and sign in with GitHub. Your handle defaults to your GitHub username.
  </Step>
  <Step title="Install the MCP server">
    Add the /vibe MCP server to your IDE config:

    <Tabs>
      <Tab title="Claude Code">
        Add to `~/.claude.json`:
        ```json
        {
          "mcpServers": {
            "vibe": {
              "command": "npx",
              "args": ["-y", "slashvibe-mcp"]
            }
          }
        }
        ```
        Then restart Claude Code.
      </Tab>
      <Tab title="Cursor">
        Add to your Cursor MCP settings (see [Cursor setup guide](/guides/ide-support)):
        ```json
        {
          "mcpServers": {
            "vibe": {
              "command": "npx",
              "args": ["-y", "slashvibe-mcp"]
            }
          }
        }
        ```
      </Tab>
      <Tab title="Other MCP IDEs">
        Any MCP-compatible IDE can run /vibe. Add to your MCP config:
        ```json
        {
          "mcpServers": {
            "vibe": {
              "command": "npx",
              "args": ["-y", "slashvibe-mcp"]
            }
          }
        }
        ```
      </Tab>
    </Tabs>
  </Step>
  <Step title="Send your first message">
    Ask Claude:
    ```
    who's online on /vibe?
    ```
    Then message someone:
    ```
    send a DM to @seth saying hello
    ```
    You'll see delivery receipts: sent, delivered, read.
  </Step>
</Steps>

## What you can do now

Once installed, try these:

| Command | What it does |
|---------|-------------|
| `who's online?` | See who's building right now |
| `DM @handle message` | Send a direct message |
| `check my messages` | Read your inbox |
| `go live` | Start a Watch Me Code session |
| `invite someone` | Generate an invite code |
