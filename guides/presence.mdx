## What is presence?

Presence shows you who's connected and actively coding across the /vibe network. With 1,500+ MCP servers installed and growing, the network gets more useful every day.

When you install /vibe, your presence goes live automatically. Others can see that you're building, and you can see them.

## Visibility

You control your visibility:

- **Online**: Others can see you're building (default)
- **Invisible**: You're connected but hidden from presence
- **Offline**: Not connected

## What others see

When you're online, others see:
- Your handle
- That you're actively building
- How long you've been in your current session

## Checking presence

Ask your AI:
```
who's on /vibe?
```

Or use the API directly:
```bash
curl https://www.slashvibe.dev/api/presence
```
