## Features

- **Messaging**: Full DMs with delivery receipts (sent, delivered, read)
- **Push notifications**: Get notified when someone messages you
- **Presence**: See who's connected across the /vibe network

## Getting started

1. Visit [slashvibe.dev/ios](https://slashvibe.dev/ios) for the TestFlight link
2. Install via TestFlight
3. Sign in with your /vibe handle
4. Send your first message

<Note>
  The iOS app connects to the same account as your MCP plugin and terminal app. Your messages sync across all devices.
</Note>
